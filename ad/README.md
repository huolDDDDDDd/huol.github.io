# 智能记账日记 - NLP驱动的财务管理助手

一个基于自然语言处理技术的现代化财务管理Web应用，让记账变得像聊天一样简单。

## 🌟 核心特性

### 1. 自然语言处理记账
- 用日常语言描述收支，AI自动解析
- 智能提取金额、分类、时间等信息
- 无需学习复杂格式，说话就能记账

### 2. 智能日记生成
- 基于记账记录和心情标签自动生成日记
- AI辅助写作，记录生活点滴
- 心情趋势分析和可视化

### 3. 数据可视化分析
- 专业的财务数据图表展示
- 收支趋势、分类占比、月度对比
- 财务健康评分系统

### 4. 预算管理
- 分类预算设置和监控
- 实时预算执行进度显示
- 超支预警和建议

## 🛠 技术架构

### 前端技术栈
- **HTML5 + CSS3 + JavaScript**: 现代Web标准
- **Tailwind CSS**: 实用优先的CSS框架
- **ECharts.js**: 专业数据可视化
- **Anime.js**: 流畅动画效果
- **P5.js**: 创意编程和动态背景

### 核心功能模块
- **NLP服务模块**: 自然语言处理集成
- **数据管理模块**: 本地存储和数据管理
- **日记模块**: 智能日记生成和管理
- **可视化模块**: 图表渲染和数据展示
- **UI交互模块**: 用户体验和动画效果

### 4. 分析报告页 (analytics.html)

  - 月度对比柱状图
- **预算管理**: 分类预算设置和监控
- **财务健康评分**: 综合评估系统

## 🎯 核心交互功能

### 自然语言记账
```
输入示例:
- "今天午餐花了35元"
- "地铁费8块钱"
- "收到工资5000元"

AI自动解析:
- 金额: 35元
- 类型: 支出
- 分类: 餐饮
- 描述: 午餐
```

### 智能日记生成
```
选择条件:
- 日期: 2026-01-02
- 心情: 4星 (还不错)
- 关键词: 工作、学习

AI生成内容:
"今天心情不错。工作上有些事情需要处理，
今天也抽时间学习了一些新知识。今天的花费情况：
- 午餐: ¥35.00
- 地铁费: ¥8.00
总支出: ¥43.00
今天是充实的一天，希望明天也能保持这样的好心情。"
```

### 数据可视化分析
- **支出分布**: 直观了解钱花在哪里
- **收支趋势**: 掌握财务变化规律
- **月度对比**: 对比不同时期的财务状况
- **预算执行**: 监控预算完成情况

```javascript
  category: "food",
  description: "午餐",
  rawText: "今天午餐花了35元",

  // ...

```

## 🎨 设计特色

### 视觉风格
- **现代简约**: 去除不必要的装饰，专注于功能
- **数据驱动**: 通过图表和可视化元素传达信息
- **情感化设计**: 结合日记功能，增加人性化元素
- **专业可信**: 财务应用需要传达安全感和专业性

### 动画效果
- **页面加载**: 元素从下方淡入，营造层次感
- **数据更新**: 数字滚动动画，增强视觉反馈
- **图表动画**: 图表数据动态绘制
- **交互反馈**: 按钮悬停时的微妙缩放和阴影变化

### 响应式设计
- **桌面端**: 充分利用屏幕空间，展示丰富的数据可视化
- **平板端**: 调整布局比例，保持功能的完整性
- **移动端**: 简化界面元素，优化触摸交互体验
