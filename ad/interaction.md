# NLP备忘录记账日记 - 交互设计

## 核心交互功能

### 1. 智能记账面板（主要交互组件）
**位置**: 主页中心区域
**功能描述**: 
- 左侧：自然语言输入框，用户可以用日常语言输入记账信息
  - 示例输入："今天午餐花了35元"、"地铁费8块"、"收到工资5000元"
- 中间：智能解析结果显示，显示NLP解析后的结构化数据（金额、类别、时间、描述）
- 右侧：快速分类选择器，提供预设的支出/收入类别（餐饮、交通、购物、工资等）

**交互流程**:
1. 用户在输入框中输入自然语言描述
2. 系统实时解析并显示结构化结果
3. 用户可手动调整分类或确认保存
4. 保存后数据自动更新到财务记录中

### 2. 日记智能生成器（主要交互组件）
**位置**: 日记页面左侧
**功能描述**:
- 顶部：日期选择器和心情评分（1-5星）
- 中间：关键词标签选择器（工作、生活、学习、运动、社交等）
- 底部：AI辅助写作区域，根据当日记账记录和选择的关键词自动生成日记草稿

**交互流程**:
1. 用户选择日期和心情评分
2. 选择相关关键词标签
3. 系统分析当日财务数据和相关标签
4. 生成日记草稿，用户可编辑和完善
5. 保存为完整日记条目

### 3. 财务数据可视化分析（主要交互组件）
**位置**: 分析页面
**功能描述**:
- 顶部：时间范围选择器（本周、本月、本季度、自定义）
- 左侧：支出分类饼图，显示各类别占比
- 右侧：收支趋势折线图，显示时间序列变化
- 底部：月度对比柱状图，显示收入vs支出对比

**交互功能**:
- 点击图表元素可查看详细数据
- 时间范围切换时图表动态更新
- 支持数据导出功能

### 4. 智能搜索与筛选器（辅助交互组件）
**位置**: 各页面顶部
**功能描述**:
- 全文搜索框，支持自然语言搜索
- 多维度筛选器：时间范围、金额范围、类别、标签等
- 智能标签建议，根据输入内容推荐相关标签

### 响应式设计
